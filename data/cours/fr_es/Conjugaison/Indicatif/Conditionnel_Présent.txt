
[font_size=30][center][b]Le conditionnel présent de l'indicatif[/b][/center][/font_size]

[center]__________[/center]

Le conditionnel présent espagnol (condicional simple) exprime des actions qui pourraient se produire en toute probabilité. Il est aussi employé dans des formules de politesse et pour exprimer des désirs.

[center][b]-- Quand utiliser le conditionnel présent de l'indicatif en espagnol ? --[/b][/center]

Le conditionnel présent est utilisé dans les cas suivants : 

  - pour les invitations
      [i][color=blue]Ex : ¿Te gustaría venir a ver una película esta noche conmigo? (Voudrais-tu venir voir un film avec moi ce soir ?)[/color][/i]

  - pour une demande polie
      [i][color=blue]Ex : ¿Podría darme un vaso de agua, por favor? (Puis-je avoir un verre d'eau, s'il vous plaît ?) [/color][/i]

  - pour exprimer un désir
      [i][color=blue]Ex : ¡Me encantaría que! (Cela me ferait plaisir!)[/color][/i]

  - pour une suggestion
      [i][color=blue]Ex : La nevera está vacía, ¿no debería ser hora de rellenarla? (Le réfrigérateur est vide, ne serait-il pas temps de le remplir ?)[/color][/i]

  - pour une supposition ou l’expression d’une probabilité
      [i][color=blue]Ex : Podría llover hoy. (Il pourrait pleuvoir aujourd'hui.) [/color][/i]

  - pour exprimer un doute ou une incertitude du passé
      [i][color=blue]Ex : ¡Pensé que estarías en casa! (Je pensais que tu serais chez toi!) [/color][/i]

  - pour exprimer le futur dans le passé (une action future considérée par rapport à une action passée)
      [i][color=blue]Ex : Albi había dicho que iría a otra parte. (Albi avait dit qu'elle irait ailleurs.) [/color][/i]

[center][b]-- Comment conjuguer les verbes espagnols au conditionnel présent de l'indicatif ? --[/b][/center]

Pour conjuguer les verbes espagnols au conditionnel présent, on remplace la terminaison du verbe conjugué au futur simple par les terminaisons suivantes (terminaisons de l'imparfait de l'indicatif des verbes ER/IR) :

   [i]yo[/i] : [b][color=red]-ía[/color][/b]
   [i]tu[/i] : [b][color=red]-ías[/color][/b]
   [i]el/ella/usted[/i] : [b][color=red]-ía[/color][/b]
   [i]nosotros[/i] : [b][color=red]-íamos[/color][/b]
   [i]vosotros[/i] : [b][color=red]-íais[/color][/b]
   [i]ellos/ellas/ustedes[/i] : [b][color=red]-ían[/color][/b]

[b]Exemple pour le verbe tomar[/b] :
   [i]yo[/i] : [b][color=blue]tomaría[/color][/b]
   [i]tu[/i] : [b][color=blue]tomarías[/color][/b]
   [i]el/ella/usted[/i] : [b][color=blue]tomaría[/color][/b]
   [i]nosotros[/i] : [b][color=blue]tomaríamos[/color][/b]
   [i]vosotros[/i] : [b][color=blue]tomaríais[/color][/b]
   [i]ellos/ellas/ustedes[/i] : [b][color=blue]tomarían[/color][/b]
