[font_size=30][center][b][color=#0000FF]Le Participe Présent en Anglais[/color][/b][/center][/font_size]

[center][color=#808080]__________[/color][/center]

Le participe présent en anglais est une forme verbale qui se termine généralement en "-ing". Il est utilisé dans plusieurs contextes, notamment pour former les temps progressifs, les adjectifs et les noms dérivés des verbes.

[font_size=24][color=#4B0082]1. Formation du participe présent[/color][/font_size]

Pour former le participe présent en anglais, on ajoute simplement "-ing" à la base du verbe. Voici des exemples avec quelques verbes courants :

[indent][b]Exemples :[/b]
[code]
to walk (marcher) -> walking (marchant)
to talk (parler) -> talking (parlant)
to eat (manger) -> eating (mangeant)
to go (aller) -> going (allant)
to write (écrire) -> writing (écrivant)
[/code][/indent]

Cependant, il y a quelques règles orthographiques à respecter :

[indent]a) Pour les verbes se terminant par une voyelle suivie d'une consonne, on double la consonne :
[code]
to run (courir) -> running (courant)
to sit (s'asseoir) -> sitting (s'asseyant)
to get (obtenir) -> getting (obtenant)
[/code]

b) Pour les verbes se terminant par un "e", on supprime le "e" avant d'ajouter "-ing" :
[code]
to make (faire) -> making (faisant)
to take (prendre) -> taking (prenant)
to write (écrire) -> writing (écrivant)
[/code][/indent]

[font_size=24][color=#4B0082]2. Utilisation du participe présent[/color][/font_size]

Le participe présent est utilisé dans plusieurs contextes en anglais. Voici les principaux usages :

[indent][b]a) Les temps progressifs[/b]
Le participe présent est utilisé avec l'auxiliaire "to be" pour former les temps progressifs, qui indiquent une action en cours.

[i]Exemple : [color=#0000FF]She is reading a book.[font_size=18](Elle est en train de lire un livre.)[/font_size][/color][/i]

[b]b) Les adjectifs[/b]
Le participe présent peut aussi être utilisé comme adjectif pour décrire une caractéristique ou une qualité.

[i]Exemple : [color=#0000FF]It was an interesting movie.[font_size=18](C'était un film intéressant.)[/font_size][/color][/i]

[b]c) Les noms dérivés des verbes (gérondifs)[/b]
Le participe présent peut être utilisé comme un nom pour désigner une action ou un état d'être.

[i]Exemple : [color=#0000FF]Swimming is fun.[font_size=18](Nager est amusant.)[/font_size][/color][/i][/indent]

[font_size=24][color=#4B0082]3. Précisions supplémentaires[/color][/font_size]

Il y a quelques particularités à noter concernant l'utilisation du participe présent en anglais :

[indent][b]a) Après certaines prépositions[/b]
Le participe présent est souvent utilisé après certaines prépositions pour indiquer une action en relation avec un nom ou un pronom.

[i]Exemple : [color=#0000FF]She is good at drawing.[font_size=18](Elle est douée pour dessiner.)[/font_size][/color][/i]

[b]b) Actions simultanées[/b]
Le participe présent peut être utilisé pour indiquer des actions qui se produisent en même temps.

[i]Exemple : [color=#0000FF]She listened to music while working.[font_size=18](Elle écoutait de la musique en travaillant.)[/font_size][/color][/i][/indent]
